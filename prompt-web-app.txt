Xây dựng web app học tập bằng React + TypeScript + Firebase/Firestore, giao diện đẹp và responsive.

Chức năng chính:
- Thư viện câu hỏi theo cấu trúc: Lớp → Môn → Chương → Bài → Câu hỏi.
- Giáo viên tạo/sửa/xóa/lọc câu hỏi, chọn câu từ thư viện.
- Giáo viên chuyển câu hỏi vào trò chơi và giao bài theo mode:
  run, pirate, blind-box, elimination, arena.
- Quản lý game assignment: xem, sửa, xóa, phát hành, thu hồi.
- Học sinh chỉ thấy bài đã phát hành, vào chơi trực tiếp trong app.
- Lưu kết quả chơi lên Firestore.
- Trang thống kê cho giáo viên: lượt chơi, điểm trung bình, theo lớp, theo mode, kết quả gần đây.

Yêu cầu:
- UI hiện đại, dễ dùng cho giáo viên và học sinh.
- Code sạch, dễ mở rộng.
- Có các trang: thư viện câu hỏi, giao trò chơi, quản lý bài giao, danh sách game cho học sinh, màn chơi game, thống kê giáo viên.
